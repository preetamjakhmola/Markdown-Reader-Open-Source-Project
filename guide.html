Getting started

Markdown Reader shows local .md files in the browser with a documentation-style layout, outline, and optional folder browsing.

1. Allow access to file URLs

This extension runs on local files (file://). Chrome must be allowed to inject the reader on those pages.

  1. Open chrome://extensions (paste into the address bar).
  2. Find Markdown Reader and click Details.
  3. Turn on Allow access to file URLs.
Required. Without this, opening a local .md file will show plain text instead of the reader.

2. Open a Markdown file

Use File → Open File (Ctrl+O) or drag a .md file into the browser. The page should render with the sidebar, table of contents, and formatted content.

3. Browse a folder of Markdown files

Open a folder URL in the address bar, for example: file:///C:/path/to/your/docs/

The extension can jump between .md files from the sidebar. Use the rail icons for Files in folder, search, and outline.

Tips

  • Collapse the sidebar with the chevron at the bottom of the left rail for more reading space.
  • Use the menu (⋯) for raw source, fullscreen, print, and about.
  • Filter headings in the outline using the search box above the table of contents.
You can reopen this page anytime from the extension’s Options (right-click the extension icon → Options), or from About inside the reader menu when viewing a file.
